bool Schedule::interferesWith ( const Schedule &  schedule) const

Checks if a schedule interferes with another. Two schedule interfere if one is partially, or completely, inside the other. E.g.: 13h-15h interferes with 14h-16h, 12h-16h, 12h-14h, 13h-15h, etc and does not with 11h-12h, 16h-17h, etc.

bool Schedule::isInvalid ( ) const

Checks if a schedule is valid or not. A schedule is invalid if the beginning date is after the ending date, or if it consists of only one "moment", this is, the beginning and the ending date is the same.

Returns
true, if the schedule is valid. false, otherwise
Schedule Schedule::mergeWith ( const Schedule &  schedule) const

Gets a schedule that is a merge of this schedule with a given one. Simplified example: 13h-15h merging with 14h-16h would return 13h-16h. 14h-15h merging with 13h-16h would return 13h-16h. Note: "interferesWith" is called; if the schedules do not interfere, the result will be only the first schedule.
