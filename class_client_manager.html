const long ClientManager::DAYS_TO_INACTIVITY = 30
static

The number of days required, without making any contract, to make a client inactive. I.e., if a client doesn't make a contract for this amount or more of days, it is labeled as inactive. E.g.: if DAYS_TO_INACTIVITY == 30 and a client doesn't make contracts for 30 days, then it is inactive.
