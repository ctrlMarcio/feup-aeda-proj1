Calculates the difference in hours from two dates. The calculation uses "dayDifference" to get the difference in days. If the difference is lower than one hour, returns 0.

E.g.: 05/05/2020 12:25.0 to 05/05/2020 13:24.59 would return 0 05/05/2020 12:25.0 to 05/05/2020 13:25.00 would return 1
